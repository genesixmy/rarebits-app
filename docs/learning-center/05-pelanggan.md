## Tujuan Modul Pelanggan

Modul ini membantu anda:
- simpan rekod pelanggan (nama, emel, telefon, alamat),
- semak sejarah pembelian,
- lihat jumlah belanja pelanggan,
- rancang follow-up jualan.

## Fungsi Utama

1. Senarai pelanggan
- Carian cepat nama/emel.
- Ringkasan jumlah pembelian dan jumlah belanja.

2. Detail pelanggan
- Profil pelanggan.
- Sejarah invois.
- Pecahan belian ikut kategori/platform.

3. Kemas kini maklumat
- Boleh sunting maklumat untuk kekalkan data tepat.

## Cara Guna Praktikal

1. Selepas invois settle, semak pelanggan tersenarai betul.
2. Pastikan emel/telefon konsisten (format sama).
3. Guna page detail untuk lihat pelanggan bernilai tinggi.
4. Export/rujuk sejarah belian bila perlu follow-up.

## Data Quality Untuk Pelanggan

1. Elak duplikasi emel/telefon dalam akaun sama.
2. Standardkan format nombor telefon.
3. Isi alamat sekurang-kurangnya satu versi yang boleh digunakan.

## Cadangan Operasi

1. Tag pelanggan berulang secara manual di luar sistem jika perlu kempen.
2. Gunakan data pelanggan sebagai rujukan servis selepas jualan.
3. Semak pelanggan yang lama tidak membeli untuk tindakan re-engagement.
