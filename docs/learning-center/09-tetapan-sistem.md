## Tujuan Modul Tetapan

Tetapan memastikan Rarebits ikut identiti bisnes dan aliran operasi anda.

Tab utama:
1. Profil
2. Kategori
3. Platform Fee
4. Brand (Invoice Settings)
5. Data Safety

## 1) Profil

Kegunaan:
- Kemas kini identiti pengguna.
- Urus avatar/maklumat asas akaun.

Amalan:
- Pastikan maklumat profil sentiasa up to date.

## 2) Kategori

Kegunaan:
- Kategori item untuk inventori dan analitik.

Amalan:
- Gunakan set kategori stabil.
- Elak duplicate category dengan ejaan berbeza.

## 3) Platform Fee Settings

Kegunaan:
- Cipta rule caj platform yang boleh dipilih dalam invois.

Konfigurasi rule:
1. Nama rule
2. Jenis fee (`percentage` atau `flat`)
3. Base kiraan (`item_subtotal`, `shipping_charged`, `total_collected`)
4. Aktif/tidak aktif

Amalan:
- Simpan rule yang masih digunakan sahaja.

## 4) Invoice Settings (Brand)

Kegunaan:
- Tetapan paparan invois untuk A4/thermal/paperang.

Elemen biasa:
1. Nama syarikat dan maklumat hubungan.
2. Logo.
3. Marketplace links.
4. QR settings.
5. Pilihan papar/hidden untuk setiap mode cetak.

Amalan:
- Uji cetak sebenar selepas ubah style.
- Kekalkan format ringkas dan mudah dibaca.

## 5) Data Safety

Kegunaan:
- Muat turun backup penuh.
- Semak rekod backup terkini.
- Akses flow restore (rujuk modul Data Safety khusus).

Amalan:
- Backup berkala walaupun operasi kecil.
