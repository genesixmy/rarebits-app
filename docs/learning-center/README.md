# Rarebits Learning Center

Panduan ini disediakan untuk pengguna Rarebits yang mahu gunakan sistem sebagai **operational truth tool** untuk bisnes kecil (contoh: mainan secondhand, curator, collector).

Dokumen ini fokus pada:
- cara guna setiap modul,
- aliran kerja harian yang praktikal,
- amalan rekod yang konsisten supaya data dashboard tepat.

## Format Learning Center

## Quick Start

1. `00-quick-start-15-minit.md`  
   Setup minimum dan aliran pertama hingga data muncul di dashboard.

## Advanced Guide

1. `12-advanced-guide.md`  
   SOP disiplin data, governance wallet, audit mingguan, monthly close.

## Rujukan Modul Lengkap

1. `01-pengenalan-dan-aliran-kerja.md`  
   Konsep asas Rarebits, struktur data, dan rutin harian/mingguan.
2. `02-dashboard-operasi.md`  
   Cara baca metrik, graf, dan buat keputusan operasi.
3. `03-inventori-dan-item.md`  
   Pengurusan item, platform iklan, tag, status, media.
4. `04-invois-dan-jualan.md`  
   Cipta invois, platform jualan invois, penghantaran, caj platform, cetak/share.
5. `05-pelanggan.md`  
   Rekod pelanggan, sejarah belian, analitik pelanggan.
6. `06-wallet-dan-resit.md`  
   Wallet, transaksi, lampiran resit, halaman resit, muat turun ikut penapis.
7. `07-reminder.md`  
   Reminder berulang, view list/kalendar, susulan tugas.
8. `08-katalog-awam.md`  
   Cipta katalog awam untuk kongsi item.
9. `09-tetapan-sistem.md`  
   Tetapan profil, kategori, brand invois, platform fee, data safety.
10. `10-data-safety-backup-restore.md`  
    Backup penuh, restore media sandbox, disaster restore, amalan selamat.
11. `11-faq-dan-troubleshooting.md`  
    Soalan lazim dan langkah semak cepat.

## Laluan Belajar Disyorkan

Untuk pengguna baru:
1. Mulakan dengan `00-quick-start-15-minit.md`.
2. Baca `01` (asas + aliran kerja).
3. Selesaikan setup awal dalam `09`.
4. Belajar `03` dan `04` dahulu (core operasi).
5. Sambung `06` (wallet/resit), kemudian `02` (dashboard).
6. Akhir sekali, amalkan backup ikut `10`.
7. Bila dah stabil, masuk `12-advanced-guide.md`.

## Matlamat Penggunaan

Jika digunakan konsisten, Rarebits membantu anda:
- tahu stok semasa dan pergerakan jualan,
- rekod cashflow masuk/keluar dengan bukti resit,
- semak prestasi platform jualan,
- rujuk sejarah pelanggan dengan cepat,
- pulih data dengan lebih selamat jika berlaku isu.
